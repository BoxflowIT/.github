# .github

Org-wide defaults for [BoxflowIT](https://github.com/BoxflowIT).

## What's here

- `profile/README.md` – Public org profile shown at github.com/BoxflowIT
- `.github/ISSUE_TEMPLATE/` – Default issue templates for all repos
- `.github/PULL_REQUEST_TEMPLATE.md` – Default PR template
- `workflow-templates/` – Reusable workflow templates available when creating new workflows
